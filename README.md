# Anon CMS Plugin List

This repository contains a list of plugins for Anon CMS.

## JSON Format

The list of plugin is stored in a JSON file.

```json
{
    "name": "plugin name",
    "author": "you plugin",
    "description": "plugin description",
    "mode": "auto",
    "url": {
        "github": "GitHub url",
        "gitee": null
    }
}
```
